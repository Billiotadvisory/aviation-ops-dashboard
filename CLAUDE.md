# Project notes for Claude Code

## Writing style preferences (Kieuanh / Billiot Advisory)

- **Never use em dashes (—) in any writing meant for the user**: posts, copy, in-chat replies, code comments, commit messages, PR bodies, dashboard text. Use commas, periods, parentheses, or sentence breaks instead.
- This applies to body text shown to the user and to user-facing content that gets shipped. Hyphens inside compound words (multi-step, single-thread, etc.) are fine.
